# my_FYP
This is a study on Physics-Guided Multimodal Sensor Data Generation and Enhancement
## Reproducibility Statement

This repository reproduces the results of the paper:

"Data Can Speak for Itself: Quality-guided Utilization of Wireless Synthetic Data" by Chen Gong，Bo Liang，Wei Gao，Chenren Xu.

We use the official implementation released by the authors, with minimal
modifications for running the experiments.
The original code is available at:
(https://github.com/pittisl/SynCheck.git)
