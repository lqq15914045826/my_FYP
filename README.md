# my_FYP

This project studies **Physics-Guided Multimodal Sensor Data Generation and Enhancement**.

## Reproducibility Statement

This repository reproduces the experimental results reported in the paper:

> **Data Can Speak for Itself: Quality-guided Utilization of Wireless Synthetic Data**  
> Chen Gong, Bo Liang, Wei Gao, Chenren Xu.

The experiments are conducted using the **official implementation** released by the authors, with **minimal modifications** required to run the code in our environment.

The original implementation is available at:  
https://github.com/pittisl/SynCheck
